# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this website, please send an email to security@yourdomain.com. 
All security vulnerabilities will be promptly addressed.

Please do not disclose security-related issues publicly until a fix has been announced.

## Security Measures
- All content is served over HTTPS
- Content Security Policy (CSP) implemented
- Regular security updates and monitoring
- Protected against XSS and CSRF attacks
